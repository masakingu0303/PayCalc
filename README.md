# 物販売上管理

##  機能説明
１件ごとに品目,単価,数量,バック,金額(税込み)の項目があり入力する　
金額=単価×数量
月の金額の合計表示
月毎に表示tableで表示
ソート日付順
私用の為、初期値は個人的によく使う項目を設定
私用の為、簡易ログイン機能

##  使用技術スタック 
* フロントエンド React (TypeScript) + Vite
* スタイリング CSS React Icons
* データ管理　json-server


##  実行環境
MacBook Air　MacOSバージョン14.5 メモリ8G
